# ML-Based-Phishing-Email-Detection# Phishing Email Detection Using Machine Learning

## Overview
This project detects phishing emails using machine learning and natural language processing techniques.

## Features
- TF-IDF text vectorization
- URL analysis
- Keyword detection
- Random Forest classifier
- Accuracy evaluation
- Confusion matrix visualization
- Model persistence

## Technologies Used
- Python
- Scikit-Learn
- Pandas
- Matplotlib

## Output
The model classifies emails as:

- Safe
- Phishing

It also generates:
- Accuracy score
- Classification report
- Confusion matrix graph

## Run

pip install -r requirements.txt

python phishing_detector.py
